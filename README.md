## Symfony site using Twig

This simple site is coded following this (https://symfonycasts.com/screencast/symfony) online tutorial on Symfony 5. It focuses on the basic features of Symfony such as :
* Route, controllers
* Annotations, wildcard routes
* Recipes, aliases
* Twig
* Debugging
* Assets
* URLs generation
* JSON API endpoint
* JS, AJAX
* Smart routes
* Service objects
* Webpack Encore
